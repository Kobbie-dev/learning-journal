create the repo on Github
Click the + icon in the top right and select "New repository"
Name it something like learning-journal
Add a description like "My daily learning notes"
Choose "Public" or "Private"
Check "Add a README file" (optional but helpful)
Click "Create repository"
Open terminal and run:
git clone https://github.com/YOUR_USERNAME/learning-journal.git
cd learning-journal
Replace username with actual github username
create an entry using .md at the end
open  in editor like VS studio and write notes or code
COMMIT AND PUSH
git add .md
git commit -m "docs: add learning notes or code"
git push origin main
Enter Username And Password
Remember to ger a personal access token and use as password when asked
